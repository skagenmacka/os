# Introduktion

Projektet omfattar ett forskningsprojekt inom operativssystem, skrivet i C och ARM64 Assembly för AArch-64  arkitekturen. Syftet med projektet är att utforska *low level* systemutveckling och förstå grundläggande komponenter för att skapa ett operativsystem från grunden. 

Systemet är inpererat av filosofin och tekniska arkitekuren hos ett Unix-liknande operativssystem samt enkelheten som finns i moderna mobila operativsystem.

Kerneln (kärnan) är designad att köras på en virtuell QEMU maskin.
